💚미디어쿼리(Media Query)

미디어쿼리는 화면 크기에 따른 각각의 속성값을 지정하여, 여러가지 화면을 구성하는 기술이다.

@media only all and (조건문){실행문}

  • @media : 미디어쿼리가 시작됨을 표시합니다.
  • only : 미디어쿼리 구문을 해석하라는 명령어입니다.(생략가능)
  • all : 미디어쿼리 구문을 해석하라는 명령어입니다.(생략가능)
    • all : 모든 미디어 유형에서 사용할 css를 정의합니다.
    • print : 인쇄장치에서 사용할 css를 정의합니다.
    • screens : 컴퓨터 스크린에서 사용할 css를 정의합니다.
    • aural : 화면을 읽어 소리로 출력해 주는 장치에서 사용할 css를 정의합니다.
    • tv : TV에서 사용할 css를 정의합니다.
    • handheld : 손에 들고 다니는 장치에서 사용할 css를 정의합니다.
    • projection : 프로젝트를 위한 css를 정의합니다.
  • and : 앞과 뒤의 조건을 나타냅니다.(생략가능)
    ex-> @media screens (min-width:400px) and (max-width:900px){실행문} //400~900사이
    ex-> @media screens (max-width:900px){실행문} //최대넓이 900 = 900이하
  • 조건문 : 해당 조건을 설정 할 수 있습니다.
  • 실행문 : 조건에 따른 실행을 설정합니다.
